# Pixel 3a OS Rebuild (google-sargo)

Build a ground-up OS image for Google Pixel 3a (sargo) using postmarketOS edge; prepare flashable images and execute flashing once device is in fastboot. Includes environment audit, build config, and background build.

- `operator`
- `app-dev`
