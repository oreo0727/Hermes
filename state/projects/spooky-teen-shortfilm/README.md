# Midnight Signals — AI Spooky Short for Teens

Develop a 5–7 minute AI-generated spooky short film for a teen audience (PG-13 vibe): suspense over gore, phone-native storytelling, clear consent/safety boundaries, and a fast, reproducible AI production pipeline (script → boards → animatics → final).

## Specialists

- `operator`
- `creative-dev`

## Project Layout

- `brief.md` for the working brief and outcomes
- `canon.md` for stable rules, tone, and continuity
- `roadmap.md` for milestones and sequencing
- `app/` for application-facing artifacts
- `game/` for gameplay, systems, and build work
- `creative/` for story, image, video, and publishing assets
- `artifacts/` for packaged outputs
- `runs/` for structured run summaries
